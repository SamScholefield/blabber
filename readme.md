blabber
=======

Web chat system with node and web sockets.


Install
-------

Install Node.js - http://nodejs.org/

Open a terminal and cd to where package.json is.

Get dependencies by typing: 

    npm install -d

Marvelous.
 
 
Run Server
----------

    node babble.js
    
Connect Client
--------------

Point a web browser at

    http://localhost:8080
